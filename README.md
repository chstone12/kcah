# Korean Cards Against Humanity.
Cards Against Humanity 한국어 버전이구요<br>
참고로 카드들은 제 아이디어 말고도 추천받은 것들이 꽤 많구요 (그리고 저도 똑바로 확인 안해서 뭐뭐있는지 잘 모름)<br>
cmd에서 pip install flask, pip install flask_socketio 한번씩 쳐야 실행되구요 <br>
네이버에다가 '내아이피' 쳐서 나온 그 아이피주스롤 <br>
http:// [여기] :80/main <br>
에다가 넣고 원하는 사람들한테 링크 주시면 됩니다. <br>
아이피주소의 도메인이 따로 있다면 뭐 당연히 그거 쓰셔도 되구요 <br>
